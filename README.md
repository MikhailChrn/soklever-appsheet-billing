# СО Клевер Биллинг

База знаний проекта **СО Клевер Биллинг** — системы биллинга для частной школы СО Клевер.

Стек: Google Sheets (хранилище) + AppSheet (UI и логика). Этап развития — «Google-прародитель», временное решение до перехода на основной продукт.

**Дедлайн запуска:** 01.06.2026.

---

## Что в репозитории

Это **долгоживущая память проекта**: бизнес-модель, архитектурные решения, регламенты, история работы. Сам код приложения живёт в AppSheet и Google Sheets — здесь только знание о нём.

## Принципы работы с базой

1. **Корневые `*.md`** редактируются только при изменении системы или решений. Не при каждой задаче.
2. **`subgeneral-backups/`** пополняется регулярно — это история состояний проекта. Старые файлы не удаляются.
3. **`briefings/` и `reports/` — append-only.** После создания не редактируются.
4. **`README.md`** обновляется при крупных вехах (запуск, переход на новый этап).
